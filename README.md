# CyberFresco
Quick and easy HTML image gallery creation.

Builds a complete set of thumbnails, webpages (with navigation), and indexes, for a folder (or folders) of images. 

C++ Builder 12 rewrite and major update. Download the free community edition here:

https://www.embarcadero.com/products/cbuilder/starter/free-download

Started in 2005, with the last release in 2008, this is a major update and rewrite.

Initial code is uploaded. Mostly functional, FTP uploading is not finished, lots of tweaks and QoL improvements to come.
